# Sensor Application #

Provide 2 GET request

1. for Humidity read
2. For oxygen read

## 5 HTTP GET ##

'/healty' for livenss prob

'/oxygen' for Oxygen reading

'/listoxygenreads' To get all oxygen readings

'/humidity' To get Humidity Reading

'/listhumidityreads' To get all humidity readings
